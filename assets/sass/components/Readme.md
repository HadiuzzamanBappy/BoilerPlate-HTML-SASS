
# components/ - UI Components (Alerts, Avatars, Badges, etc.)

This folder contains styles for reusable UI components that form the building blocks of your application.

## Components Included

- **Alert**: Styles for alert messages (success, error, warning, info).
- **Avatar**: Styles for user avatars (circular, square, etc.).
- **Badge**: Styles for badges used to display notifications or status indicators.
- **Button**: 
  - **Icon Buttons**: Buttons with icons.
  - **Toggle Buttons**: Buttons that toggle states.
  - **Button Groups**: Groups of buttons styled together.
- **Card**: Styles for card components, including layout and appearance.
- **Checkbox**: Styles for custom checkboxes.
- **Chip**: Styles for chips or tags.
- **Container**: Styles for layout containers.
- **Dialog**: Styles for modal dialogs or pop-ups.
- **FAB (Floating Action Button)**: Styles for floating action buttons.
- **Inputs**: Styles for input fields (text, password, etc.).
- **Labels**: Styles for form labels.
- **List Item**: Styles for list items (e.g., navigation menus, dropdowns).
- **Progress**: Styles for progress bars and indicators.
- **Radio**: Styles for custom radio buttons.
- **Rating**: Styles for star or number-based rating systems.
- **Toggle**: Styles for toggle switches.
- **Tooltip**: Styles for tooltips that provide additional information.

## Usage:
Each component is designed to be modular and reusable. Import and use them where required in your project to ensure consistent UI/UX.
